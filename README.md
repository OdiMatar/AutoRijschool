# Autorijschool Odai

Pure React/Vite website voor Autorijschool Odai.

## Starten

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Deze versie gebruikt geen Laravel, Herd, PHP, Composer, Blade, migrations of database. Alle demo-data staat lokaal in de React-app.
